# How to build HeliPort

Compiling HeliPort is extremely easy, let's get right into it.

# Requirements

  • Basic (for compiling)
    • macOS 10.12+
    • Xcode Command Line Tools
    • CocoaPods
  • Advanced (for development)
    • All requirements from Basic +
    • Xcode full version (recommended)
    • SwiftLint (Required)

# Installing

  • Xcode Command Line Tools

    xcode-select --install
sudo xcode-select --switch /Library/Developer/CommandLineTools

# Building

Run the following commands in Terminal:

git clone --depth=1 https://github.com/OpenIntelWireless/HeliPort.git
cd HeliPort
pod install
xcodebuild -workspace HeliPort.xcworkspace -scheme HeliPort -configuration Release -derivedDataPath . CODE_SIGN_IDENTITY="" CODE_SIGNING_REQUIRED=NO
open ./Build/Products/Release
